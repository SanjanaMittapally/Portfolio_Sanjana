# Sanjana Mittapally - Portfolio

This is a statically exported Next.js portfolio (App Router + Tailwind).

## Local development

```bash
npm install
npm run dev
```

## Build (static export)

```bash
npm ci
npm run build
```

The generated site is placed in:

`out/`

## Content source

All portfolio data is defined in:

`src/content/resume.ts`

## Files you should add

For the header/hero “Download Resume” link to work, add your PDF at:

`public/resume.pdf`
